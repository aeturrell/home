I've been trying to get the GNU General Scientific Library working with Xcode, the Mac integrated developer environment. These are the steps I took:

 First use a package manager to install the GNU scientific library from the command line. I use MacPorts:
  sudo port install gsl

Next, open up your Xcode C++ project. Click on the target, and then navigate to "All" and "Combined" on the "Build Settings" tab.

Assuming your gsl distribution is installed in /opt/local/ then you need to add

/opt/local/include
to the "Header Search Paths" option and

/opt/local/lib
to "Library Search Paths". Finally, you need to add some Linker Flags to let the compiler know you're linking gsl. This option is found under "Other Linker Flags" where you should add

-lgsl -lgslcblas


The final step is to use include on whatever you need from the GNU Scientific Library in your code.
